# glab-axi

AXI-compliant wrapper around `glab`.

Current slice: CLI shell plus `api`, `ci list/get/status/run/create/run-trig/trace`, `run` alias, `pipeline` and `workflow` aliases, `issue list/view/create/note/comment/update/close/delete/reopen/open/subscribe/unsubscribe`, `label list/create`, `pr` and `mr list/view/create/merge`, `project view/search/find/lookup`, `release list/view`, `repo list/view`, `search repos`, `setup hooks`, and builtin `update`.

# glab-axi

AXI-compliant wrapper around `glab`.

Current slice: CLI shell plus `api`, `ci list/get/status/run/run-trig`, `pipeline` alias, `issue list/view/create/note/update/close/delete`, `label list`, `mr list/view/create/merge`, `project view/search`, `release list`, `repo view`, `search repos`, `setup hooks`, and builtin `update`.

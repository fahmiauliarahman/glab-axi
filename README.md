# glab-axi

AXI-compliant wrapper around `glab`.

Current slice: CLI shell plus `api`, `ci list/get/status/run/run-trig/trace`, `pipeline` alias, `issue list/view/create/note/comment/update/close/delete/reopen/open`, `label list`, `mr list/view/create/merge`, `project view/search/find/lookup`, `release list`, `repo view`, `search repos`, `setup hooks`, and builtin `update`.

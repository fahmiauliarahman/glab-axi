# glab-axi

AXI-compliant wrapper around `glab`.

Current slice: CLI shell plus `api`, `ci list/get/status/run`, `pipeline` alias, `issue list/view/create/note/update/close`, `label list`, `mr list/view/create`, `project view/search`, `release list`, `repo view`, `search repos`, `setup hooks`, and builtin `update`.
